Decimal

O tipo de base decimal é um sistema numérico que usa dez dígitos, de 0 a 9. Cada posição de um número representa uma potência de 10, aumentando da direita para a esquerda. É o sistema de numeração mais utilizado em contagens e cálculos cotidianos.

Binario

É um sistema numérico que utiliza apenas dois dígitos, 0 e 1. Cada posição representa uma potência de 2, aumentando da direita para a esquerda. É amplamente usado em sistemas digitais e computação.

Octal

Sistema numérico que usa oito dígitos, de 0 a 7. Cada posição corresponde a uma potência de 8, aumentando da direita para a esquerda. É útil para representar números binários de forma mais compacta.

Hexadecimal

Sistema numérico que utiliza dezesseis dígitos, de 0 a 9 e de A a F. Cada posição representa uma potência de 16, aumentando da direita para a esquerda. É comum na programação e no funcionamento interno dos computadores.
